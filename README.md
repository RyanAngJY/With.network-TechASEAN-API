# APEIRO Connector

## Deploying on AWS:
- Remember to set the environment variables for DB_DATABASE, DB_HOST, DB_PASS, DB_USER, 
NODE_ENV.

## To run locally:
```
nvm use
npm i
npm start
```

### To Deploy on Elastic Beanstalk

Git commit the code to master branch and push.

Run:

```
eb deploy
```
